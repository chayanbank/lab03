การใช้งาน Grid ใน Bootstrap

Post by Chayan Suratruangchai, Feb 5, 2017.
Grid Bootstrap

เป็น Layout ที่คอยควบคุมวัตถุต่างๆ บนหน้าเว็บไซต์ให้แสดงผลได้อย่างถูกต้องตามหลัก box layout จะส่งผลดีต่อการเขียนหนึ่งหน้าเว็บไซต์ แต่สามารถใช้ร่วมกับอุปกรณ์ทุกขนาดหน้าจอ ซึ่ง grid นั้นจะใช้ <div> ในการวาง layout และมีการไปเรียก class ที่ Bootstrap ได้กำหนดไว้แล้ว Grid system ของ Bootstrap เอง แบ่งออกเป็น 12 คอลัมน์ ถ้าให้เห็นภาพง่ายขึ้นก็น่าจะต้องบอกว่า เหมือนกับตาราง 1 แถว มี 12 คอลัมน์ ดังภาพด้านล่าง




ซึ่งหน่วยที่เป็น 12 นี้ เราสามารถเฉลี่ยแต่ละส่วนได้ตามความต้องการ อย่างเช่น

การใช้งาน Form ใน Bootstrap

Post by Chayan Suratruangchai, Feb 5, 2017.
Form Bootstrap

Form คือการให้ผู้ใช้กรอกข้อมูลลงใน Field แล้วกดส่งค่า เพื่อจะเก็บค่าของข้อมูลบางอย่าง


การจัดรูปแบบ Bootstrap มี 3 ประเภท :

  • รูปแบบแนวตั้ง (นี้เป็นค่าเริ่มต้น)
  • รูปแบบแนวนอน
  • รูปแบบอินไลน์ (บรรทัดเดียวกัน)
